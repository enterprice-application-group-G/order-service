My Contribution - Order Service

I developed read endpoints for the Order Service.

What I added:
- Added GET /orders to return all saved orders.
- Added GET /orders/{orderId} to return one order by its ID.
- Added OrderNotFoundException for orders that do not exist.
- Updated the global exception handler so missing orders return 404 Not Found.
- Added service tests for listing orders and finding orders by ID.
- Added controller tests for the new GET endpoints and the 404 response.
- Updated the README file with the new Order Service features.

Tested with:
.\mvnw.cmd test
